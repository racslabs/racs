.. _CLOSE:

CLOSE
=====

**Description:**
Closes an existing audio stream.


Arguments
---------

+-------------+------------+---------------------------------------------------------------+
| Name        | Type       | Description                                                   |
+=============+============+===============================================================+
| stream_id   | ``string`` | ASCII ID of the audio stream.                                 |
+-------------+------------+---------------------------------------------------------------+


Dependencies
------------
- ``NONE``


Output
------

+----------+--------------------------------------------+
| Type     | Description                                |
+==========+============================================+
| ``null`` | N/A                                        |
+----------+--------------------------------------------+

Example
-------

.. code-block:: none

   CLOSE 'chopin'
