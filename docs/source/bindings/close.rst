.. _SCM_CLOSE:

CLOSE
=====

**Description:**
Closes an existing audio stream.


Arguments
---------

+-------------+------------+---------------------------------------------------------------+
| Name        | Type       | Description                                                   |
+=============+============+===============================================================+
| stream_id   | ``String`` | ASCII ID of the audio stream.                                 |
+-------------+------------+---------------------------------------------------------------+

Output
------

+----------+--------------------------------------------+
| Type     | Description                                |
+==========+============================================+
| ``nil``  | N/A                                        |
+----------+--------------------------------------------+

Example
-------

.. code-block:: scheme

    ;; SCM expression
    (close "chopin")

Using ``EVAL``:

.. code-block:: none

    EVAL '(close \"chopin\")'
