.. _SCM_OPEN:

OPEN
====

**Description:**
Opens an existing audio stream.


Arguments
---------

+-------------+------------+---------------------------------------------------------------+
| Name        | Type       | Description                                                   |
+=============+============+===============================================================+
| stream_id   | ``String`` | ASCII ID of the audio stream.                                 |
+-------------+------------+---------------------------------------------------------------+


Output
------

+----------+--------------------------------------------+
| Type     | Description                                |
+==========+============================================+
| ``nil``  | N/A                                        |
+----------+--------------------------------------------+

Example
-------

.. code-block:: scheme

    ;; SCM expression
    (open "chopin")

Using ``EVAL``:

.. code-block:: none

    EVAL '(open \"chopin\")'
