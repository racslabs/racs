.. _SCM_CREATE:

CREATE
======

**Description:**
Creates a new audio stream.


Arguments
---------

+-------------+-------------+---------------------------------------------------------------+
| Name        | Type       | Description                                                   |
+=============+=============+===============================================================+
| stream_id   | ``String``  | ASCII ID of the audio stream.                                 |
+-------------+-------------+---------------------------------------------------------------+
| sample_rate | ``Integer`` | Output sample rate (Hz).                                      |
+-------------+-------------+---------------------------------------------------------------+
| channels    | ``Integer`` | Number of output channels.                                    |
+-------------+-------------+---------------------------------------------------------------+
| bit_depth   | ``Integer`` | Bits per sample.                                              |
+-------------+-------------+---------------------------------------------------------------+

Output
------

+----------+--------------------------------------------+
| Type     | Description                                |
+==========+============================================+
| ``null`` | N/A                                        |
+----------+--------------------------------------------+

Example
-------

.. code-block:: scheme

    ;; SCM expression
    (create "chopin" 44100 2 16)

Using ``EVAL``:

.. code-block:: none

    EVAL '(create \"chopin\" 44100 2 16)'
