.. _PING:

PING
====

**Description:**
Checks the connectivity and responsiveness of the server.

Dependencies
------------
- ``NONE``

Output
------

+------------+--------------------------------------------+
| Type       | Description                                |
+============+============================================+
| ``string`` | Always returns the string ``'PONG'``.      |
+------------+--------------------------------------------+

Example
-------

.. code-block:: bash

   PING
