CREATE
======

**Description:**
Creates a new audio stream.


Arguments
---------

+-------------+------------+---------------------------------------------------------------+
| Name        | Type       | Description                                                   |
+=============+============+===============================================================+
| stream_id   | ``string`` | ASCII ID of the audio stream.                                 |
+-------------+------------+---------------------------------------------------------------+
| sample_rate | ``int``    | Output sample rate (Hz).                                      |
+-------------+------------+---------------------------------------------------------------+
| channels    | ``int``    | Number of output channels.                                    |
+-------------+------------+---------------------------------------------------------------+
| bit_depth   | ``int``    | Bits per sample.                                              |
+-------------+------------+---------------------------------------------------------------+


Dependencies
------------
- ``NONE``


Output
------

+----------+--------------------------------------------+
| Type     | Description                                |
+==========+============================================+
| ``null`` | N/A                                        |
+----------+--------------------------------------------+

Example
-------

.. code-block:: none

   CREATE 'chopin' 44100 2 16
